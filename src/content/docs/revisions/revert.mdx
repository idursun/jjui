---
title: Revert Operation
description: Revert selected revisions against a target
---

Revert creates inverse changes from selected revisions and applies them relative to a target.

Press `shift+r` in the Revisions view to enter revert mode.

## Target modes

- `o`: onto (destination)
- `a`: after
- `b`: before
- `i`: insert
- `t`: open target picker

## Apply and cancel

- `enter`: apply
- `alt+enter`: force apply
- `esc`: cancel

## Target picker

The target picker uses the shared `revisions.target_picker` bindings:

- `up`/`down`: move
- `tab` / `shift+tab`: autocomplete
- `enter`: apply
- `alt+enter`: force apply
- `esc`: cancel
