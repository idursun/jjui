---
title: Set Parents Operation
description: Rewrite parents of a selected revision
---

Set Parents lets you choose parent revisions for the currently selected revision.

Press `shift+m` in the Revisions view to enter set-parents mode.

## Workflow

1. Select a revision and press `shift+m`.
2. Move through candidate parent revisions.
3. Toggle selection with `space`.
4. Press `enter` to apply.

## Keybindings

- `space`: toggle select parent
- `enter`: apply
- `esc`: cancel
- `f`: ace jump
- `@`: jump to working copy
- `j`/`k`, `pgup`/`pgdown`: navigation
