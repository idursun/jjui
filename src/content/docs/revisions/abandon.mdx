---
title: Abandon Operation
description: Remove selected revisions interactively
---

Abandon removes selected revisions from history.

Press `a` in Revisions to enter abandon mode.

## Workflow

1. Select revisions with `space` (optional for multi-select).
2. Press `a` to open abandon mode.
3. Optionally refine selection in abandon mode.
4. Press `enter` to apply (`alt+enter` to force apply).

## Abandon mode keys

- `space`: toggle selected revision
- `s`: select descendants of current revision
- `f`: ace jump
- `@`: jump to working copy
- `j`/`k`, `pgup`/`pgdown`: navigation
- `esc`: cancel
- `enter`: apply
- `alt+enter`: force apply
