---
title: Abandon Operation
description: Remove selected revisions interactively
---

Abandon removes selected revisions from history.

<img alt="Abandon" src="/jjui/images/abandon.png" />

Press `a` in Revisions to enter abandon mode. If revisions were already selected with `space`, abandon mode opens pre-populated with them. You can then refine the set further before applying.

## Abandon mode keys

- `space`: toggle selected revision
- `s`: select the highlighted revision and all its descendants — a quick way to abandon a whole chain without selecting each revision individually
- `f`: ace jump
- `@`: jump to working copy
- `j`/`k`, `pgup`/`pgdown`: navigation
- `enter`: apply
- `alt+enter`: force apply
- `esc`: cancel
