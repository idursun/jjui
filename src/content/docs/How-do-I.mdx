### How do I edit files with conflicts?

Move to a revision with conflicts, then:

- Press `l` to open details.
- Press `space` to select files.
- Run `$ vim $checked_files` to edit selected files.

### How do I create a new change directly on the current revision?

- Run `: new -A $commit_id`

### How do I create a mega merge?

- Press `space` to select revisions.
- Run `: new all:$checked_commit_ids`.

### How do I squash specific files from multiple revisions?

- Press `space` to select source revisions.
- Open details with `l` on one selected revision.
- Press `space` to select files.
- Run `: squash --from $checked_commit_ids --into @ $checked_files`.
