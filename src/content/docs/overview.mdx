### Core Features
* [Revisions](/features/revisions): Main view for exploring and managing revisions
* [Preview](/features/preview): View diffs and details of commits and files (with mouse-aware scrolling and auto placement)
* [Details](/features/details): Explore files in a commit, with diff, split, and restore capabilities
* [Oplog](/features/oplog): View the operation log of your repository
* [Command Execution](/features/command-execution): Execute shell and jj commands directly from `jjui`
* [Custom Commands](/features/custom-commands): Define your own commands with custom keybindings, key sequences, and Lua scripting
* [Fuzzy File Finder](/features/fuzzy-file-finder): Fuzzy find on all files and explore changes on them.
* [Ace Jump](/features/ace-jump): Quickly move between revisions using fast key strokes.
* [Exec](/features/exec): Run interactive jj and shell commands
* [Flash Messages](/features/flash-messages): See command output and errors in the UI
* [Bookmarks](/features/bookmarks): Manage bookmarks with a pop-up menu
* [Git](/features/git): Run git commands from a pop-up menu

### Revision Operations
* [Rebase](/operations/rebase): Interactive rebase of commits
* [Absorb](/operations/absorb): Automatically absorb changes into commits
* [Abandon](/operations/abandon): Remove commits from your history
* [Duplicate](/operations/duplicate): Copy revisions with interactive UI
* [Squash](/operations/squash): Squash multiple revisions into one
* [Evolog](/operations/evolog): Explore and restore changes from the evolution log
* [Inline Describe](/operations/inline-describe): Edit revision descriptions inline
* [Bookmark](/operations/bookmark): Set or move bookmarks on revisions

### Customization
* [Configuration](/customization/configuration): How to configure JJUI to your needs
* [Command Line Options](/customization/command-line-options): Customize behavior with command line flags
* [Leader Key](/customization/leader-key): Create powerful mnemonic shortcuts with nested keymaps
* [Themes](/customization/themes): Customize the appearance of the UI with detailed styling options
* [Tracing](/customization/tracing): Highlight lanes and dim out-of-lane revisions
