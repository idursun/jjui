### Core Features
* [Revisions](/jjui/features/revisions): Main view for exploring and managing revisions
* [Preview](/jjui/features/preview): View diffs and details of commits and files (with mouse-aware scrolling and auto placement)
* [Details](/jjui/features/details): Explore files in a commit, with diff, split, and restore capabilities
* [Oplog](/jjui/features/oplog): View the operation log of your repository
* [Command Execution](/jjui/features/command-execution): Execute shell and jj commands directly from `jjui`
* [Custom Commands](/jjui/features/custom-commands): Define your own commands with custom keybindings, key sequences, and Lua scripting
* [Fuzzy File Finder](/jjui/features/fuzzy-file-finder): Fuzzy find on all files and explore changes on them.
* [Ace Jump](/jjui/features/ace-jump): Quickly move between revisions using fast key strokes.
* [Exec](/jjui/features/exec): Run interactive jj and shell commands
* [Flash Messages](/jjui/features/flash-messages): See command output and errors in the UI
* [Bookmarks](/jjui/features/bookmarks): Manage bookmarks with a pop-up menu
* [Git](/jjui/features/git): Run git commands from a pop-up menu

### Revision Operations
* [Rebase](/jjui/operations/rebase): Interactive rebase of commits
* [Absorb](/jjui/operations/absorb): Automatically absorb changes into commits
* [Abandon](/jjui/operations/abandon): Remove commits from your history
* [Duplicate](/jjui/operations/duplicate): Copy revisions with interactive UI
* [Squash](/jjui/operations/squash): Squash multiple revisions into one
* [Evolog](/jjui/operations/evolog): Explore and restore changes from the evolution log
* [Inline Describe](/jjui/operations/inline-describe): Edit revision descriptions inline
* [Bookmark](/jjui/operations/bookmark): Set or move bookmarks on revisions

### Customization
* [Configuration](/jjui/customization/configuration): How to configure JJUI to your needs
* [Command Line Options](/jjui/customization/command-line-options): Customize behavior with command line flags
* [Leader Key](/jjui/customization/leader-key): Create powerful mnemonic shortcuts with nested keymaps
* [Themes](/jjui/customization/themes): Customize the appearance of the UI with detailed styling options
* [Tracing](/jjui/customization/tracing): Highlight lanes and dim out-of-lane revisions
