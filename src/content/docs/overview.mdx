### UI Segments

- [Revisions](/jjui/revisions): Main view for exploring and managing revisions.
- [Git](/jjui/git): Run common git operations.
- [Bookmarks](/jjui/bookmarks): Manage bookmarks.
- [Undo and Redo](/jjui/undo-redo): Navigate and apply undo/redo entries.
- [Revset](/jjui/revset): Edit and apply revset expressions.
- [Oplog](/jjui/oplog): Browse repository operation history.
- [Command Execution](/jjui/command-execution): Run `jj` and shell commands from the UI.
- [Command History](/jjui/command-history): Browse and reuse recent `:` and `$` commands.
- [Preview](/jjui/preview): View diffs and details for revisions, files, and operations.
- [Details](/jjui/details): Inspect and operate on files in a revision.
- [Diff Viewer](/jjui/diff-viewer): Full-screen diff viewer with scrolling and word wrap.
- [Fuzzy File Finder](/jjui/fuzzy-file-finder): Search files and inspect related changes.
- [Ace Jump](/jjui/ace-jump): Jump quickly between revisions.
- [Flash Messages](/jjui/flash-messages): Inline command feedback and errors.

### Revisions Operations

- [Rebase](/jjui/revisions/rebase): Move revisions relative to another target.
- [Absorb](/jjui/revisions/absorb): Absorb working-copy changes into a revision.
- [Abandon](/jjui/revisions/abandon): Remove selected revisions.
- [Duplicate](/jjui/revisions/duplicate): Copy selected revisions.
- [Squash](/jjui/revisions/squash): Combine revisions.
- [Revert](/jjui/revisions/revert): Revert selected revisions relative to a target.
- [Set Parents](/jjui/revisions/set-parents): Rewrite parents of the selected revision.
- [Evolog](/jjui/revisions/evolog): Inspect evolution history.
- [Inline Describe](/jjui/revisions/inline-describe): Edit descriptions inline.

### Customization

- [config.toml](/jjui/customization/config-toml): Configure runtime behavior in `config.toml`.
- [config.lua](/jjui/customization/config-lua): Startup-time programmable configuration via `setup(config)`.
- [Actions and Bindings](/jjui/customization/actions-and-bindings): Customize behavior with `[[actions]]` and `[[bindings]]`.
- [Lua Scripting](/jjui/customization/lua-scripting): Build workflows with the built-in Lua API.
- [Themes](/jjui/customization/themes): Theme and color customization.
- [Command Line Options](/jjui/customization/command-line-options): Startup/runtime flags.
