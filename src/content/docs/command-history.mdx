---
title: Command History
description: "Reuse and manage command history for : and $"
---

jjui stores command history for both `:` (`exec_jj`) and `$` (`exec_sh`) commands.

Press `shift+w` to open the in-app command history view.

## Keybindings

- `j`/`k` or arrows: move
- `d`: delete selected history entry
- `shift+w` or `esc`: close

This view helps you quickly rerun and clean up stored command entries.
